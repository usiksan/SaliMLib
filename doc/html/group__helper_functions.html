Detailed Description

helper inline functions are simplify some operations in the library

Helper functions are entirely represented by inline functions and are used to simplify source code representation of some operations inside the library. Of course, they can also be used in user code. Almost every one of these functions has its counterpart in the standard C/C++libraries
