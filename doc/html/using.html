Using SaliMLib

To use SaliMLib, you include It in your project, as described in the SaliMLib in-project integration section. Then you build your program as a multitasking program using the tools provided by the SaliMLib library.

A traditional example of the "Hello world" program for embedded systems is the led flashing program.

#include "SaliMCore.h"
extern "C" {
//System tick handler
void SysTick_Handler(void)
{
smTickCount++; // Increment of the system tick counter
}
int main(void)
{
//All initialisation
...
//Initialisation of SaliMLib
smInit(100);
//Main loop
while(true) {
//Led is on
LedOn();
//Wait for appropriate time
smWaitTick(300); //Here is switch to execution of other task
//Led is off
LedOff();
//Wait for appropriate time
smWaitTick(300); //Here is switch to execution of other task
}
}
}
